This is a simple web application built using Flask that helps manage events.
Users can log in, access a dashboard, and create or manage events through a basic interface

app/
 ├─ routes/        # auth, dashboard, events logic
 ├─ templates/     # HTML templates
 ├─ static/        # CSS files
 ├─ models.py      # database models
run.py             # app entry point
config.py          # app configuration
requirements.txt   # dependencies

Tech Stack
Backend: Python (Flask)
Frontend: HTML, CSS (Jinja templates)
Database: SQLite

User authentication (login & registration)

Dashboard view after login

Create, view, edit, and manage events

Basic layout with reusable templates

Simple and clean styling

SQLite database for local development

also notifications of the deadline and events.
